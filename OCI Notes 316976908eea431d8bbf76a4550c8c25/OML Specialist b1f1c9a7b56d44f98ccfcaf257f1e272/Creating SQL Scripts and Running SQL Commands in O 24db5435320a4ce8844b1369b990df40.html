Creating SQL Scripts

Scratchpad

  • OML scratchpads are used to develop, run, import and export SQL scripts
  • simplifies SQL and PL/SQL development
  • uses a direct JDBC connection to a database
  • visualization options

SQL Scripts

  • a set of SQL statements
  • can be saved and exported as a JSON and shared with other users (they can import)
  • starts with %script, ends with /

Restrictions on SQL Commands

  • for the sake of security and performance, some SQL commands are restricted

Restrictions for Database Options

  • features present in Oracle Database Enterprise Edition are not available in ADW

Restrictions for Database Initialization Parameters

  • no need to send in initialization parameters to start your services
  • you can modify these parameters if you want

Connection Groups

  • is a collection of database connections
  • known as a Zeppelin interpreter set
  • you can stop one or more connections that are listed under a connection group
  • global connection group is created automatically on provisioning of a new database
    • includes compute resource and connection group definition
