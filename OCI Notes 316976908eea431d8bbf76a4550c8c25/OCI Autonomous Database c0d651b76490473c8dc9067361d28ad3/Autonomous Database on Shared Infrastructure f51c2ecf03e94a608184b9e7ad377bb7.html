Auto Scaling

  • Autoscaling is by average OCPUs, rather than baseline number of OCPUs (without autoscaling)

Creating an ADB Clone

  • you can clone metadata (objects and tables)/ full adb, from backup or live running ADB
  • you can configure an ADB to use private endpoints by cloning from an ADB with public endpoints
  • you can create a refreshable (read-only) clone

Creating Alarms and Events

  • create topic → create subscription in topic → create alarm → set alarm destination to topic → anyone with subscription to topic will receive notification when event is triggered

Backups

  • automatic, retention period 60 days, point-in-time restore
  • manual backup
  • map to object storage bucket → create database credential (if needed) in db → set default credential to credential you created

Autonomous Data Guard

  • to ensure resilience, standby instance will be on a different AD
  • recovery time objective RTO, recovery point objective RPO
